# ThinkReview - Version 2.2.3 Release Notes

**Release Date:** 25 April 2026

---

## 🎯 What's New

### More reliable ThinkReview button after install and updates

We tightened how the extension starts up in the background so the ThinkReview action shows up more reliably on **first install** and after the browser **auto-updates** the extension, including a smarter retry when the page is still loading.

### GitLab + Ollama: clearer CORS help text in dark mode

On GitLab merge requests, the inline guidance for **Ollama and CORS** (shown when you are working with Ollama locally) is easier to read in **dark theme**. Host styles that could make code snippets hard to see are no longer able to override that panel.

### Lighter in-product demo

The animated demo in the product uses a **smaller GIF file**, so pages load a bit less data while the experience stays the same.
