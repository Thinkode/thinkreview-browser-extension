# ThinkReview - Version 2.2.3 Release Notes

**Release Date:** 24 April 2026

---

## 🎯 What's New

### More reliable ThinkReview on the merge request page ✅

Some installs and browser updates could leave the ThinkReview controls missing until you refreshed the page. This build improves how the extension starts up and waits for the host page to finish loading, so the button and review entry points show up more consistently—especially right after an update or on slower, heavily dynamic pages.
