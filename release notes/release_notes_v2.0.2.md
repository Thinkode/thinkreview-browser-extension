# ThinkReview - Version 2.0.2 Release Notes

**Release Date:** 17 March 2026

---

## 🎯 What's New

### Firefox Support is Here 🦊

ThinkReview now includes Firefox compatibility updates so you can use the extension more reliably across browsers.

- **Firefox-ready packaging:** Tagged releases now include a Firefox extension build.
- **Improved module loading:** Internal loading behavior was updated to work correctly in Firefox extension contexts.
- **Smoother UI behavior:** Core review panel components now load more consistently in Firefox.

---

### GitHub Enterprise Server (GHES) Support 🏢

ThinkReview now better supports self-hosted GitHub Enterprise environments.

- **Custom GHES domains in settings:** You can add your GitHub Enterprise domain from the GitHub integration section in extension settings.
- **Improved PR page detection:** ThinkReview can now detect pull request pages more reliably on enterprise/custom domains.
- **Better permission handling:** Domain access and script matching were improved for GitHub Enterprise hosts.

---

## 🔧 Improvements

- Updated GitHub integration messaging in settings to clarify GitHub.com vs GHES behavior.
- Improved cross-platform compatibility logic for GitHub and enterprise-hosted instances.
