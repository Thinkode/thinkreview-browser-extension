# ThinkReview - Version 2.2.5 Release Notes

**Release Date:** 29 April 2026

---

## 🎯 What's New

### Sign in from the review panel via the ThinkReview portal 🔐

When you need to sign in before running a review, the integrated panel now guides you to the ThinkReview portal in a new tab—no separate Google-only sign-in step in the panel. The extension opens the portal safely from the background script, and the on-page prompt is clearer about what to do next.

### Optional announcements in the review panel 📣

Important updates can appear as a small banner at the top of the review panel. You can dismiss a message when you have seen it; your choice is remembered for that announcement so it does not keep coming back.

### Agent reviews start sooner and keep working if you step away ⚡

Custom agent reviews are kicked off in parallel with the main review flow, so agent tabs tend to populate faster. While agents are running, the panel explains that you may close it and come back shortly—the work continues in the cloud.

### Clearer summaries in the integrated panel 📋

When the AI writes the main review summary as a numbered list, the panel now shows it as an easy-to-scan bullet list so host-site styling is less likely to hide the structure.

### Reviews start when you choose (manual-first) 🖱️

Automatic review on page load is turned off in this build so reviews begin when you open ThinkReview or use the trigger on the page. The “Start review automatically” controls are hidden in the extension popup for now to match that behavior.

### Smoother experience when moving between pull requests 🧭

Switching to another pull request or merge request clears the right parts of the panel state without wiping your context when you simply navigate away from a PR page, so returning to ThinkReview on a new change is more predictable.

### Consistent “ThinkReview” naming on the page 🏷️

Fallback and in-page labels that still said “AI Review” now read **ThinkReview**, matching the rest of the product.
