# ThinkReview - Version 2.0.3 Release Notes

**Release Date:** 21 March 2026

---

## 🎯 What's New

### New Platform Setup Hub 🧭

Getting started is now much easier with a redesigned setup experience in the extension popup.

- **Single platform home:** GitHub, GitLab, Azure DevOps, and Bitbucket are now organized into a cleaner setup hub.
- **Per-platform setup screens:** Each platform now has a dedicated guided flow with clearer steps.
- **Live setup status badges:** You can now quickly see what is already ready and what still needs setup.

---

### Redesigned AI Provider Selection 🤖

Choosing between ThinkReview Cloud and Ollama is now clearer and faster.

- **Card-based provider selection:** Cloud and Ollama are presented as easy-to-scan option cards.
- **Cleaner settings flow:** AI provider configuration is grouped in a collapsible section for less clutter.
- **Better visual feedback:** Selected provider states and related options are easier to understand at a glance.

---

## 🔐 Reliability & Security Improvements

- **Safer extension opening flow:** Requests to open the extension from the web app are now validated against trusted origins.
- **Rate-limited open requests:** Added protections to prevent repeated or abusive open-extension requests.
- **More intentional popup behavior:** The extension now follows a clearer, user-triggered open flow after web app interactions.

---

## 🔧 Additional Improvements

- Added platform-specific **"Not working?"** and **setup guide** shortcuts in each setup view.
- Improved support action buttons in the popup footer (help, bug report, privacy FAQ).
- Refined Azure DevOps Server setup presentation for easier onboarding.
