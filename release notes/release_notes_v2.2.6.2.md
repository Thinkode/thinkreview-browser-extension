# ThinkReview - Version 2.2.6.2 Release Notes

**Release Date:** 22 May 2026

---

## 🎯 What's New

### Sign in from the review panel via the ThinkReview portal 🔐

When you need to sign in before running a review, the integrated panel opens the ThinkReview portal in a new tab with a clearer on-page prompt, so you can complete sign-in without a separate Google-only step inside the panel. (Thank you @JElsheikh!)

### Optional announcements in the review panel 📣

Product news and updates can appear as a small banner above the review details. You can dismiss a message when you have read it, and the extension remembers that choice for that announcement so it does not keep interrupting you. (Thank you @JElsheikh!)

### Agent reviews start sooner ⚡

Agent-style reviews begin fetching in parallel with the main review, so agent tabs tend to fill in faster while you stay on the same pull request or merge request. (Thank you @JElsheikh!)

### Clearer summaries in the integrated panel 📋

When the AI writes the main review summary as a numbered list, the panel shows it as a simple bullet list so host-site styling is less likely to hide the structure. (Thank you @JElsheikh!)

### Reviews start when you choose 🖱️

Automatic review on page load is turned off so a review begins when you open ThinkReview or use the on-page trigger. Extension popup controls for “start automatically” are hidden for now so what you see matches how reviews actually start. (Thank you @JElsheikh!)

### Smoother experience when switching pull requests 🧭

Moving between different pull requests or merge requests clears the right parts of the panel state without wiping useful context when you simply leave a PR page, so opening ThinkReview on another change feels more predictable. (Thank you @JElsheikh!)

### Consistent “ThinkReview” naming on the page 🏷️

Fallback and in-page labels that still said “AI Review” now read **ThinkReview**, matching the rest of the product. (Thank you @JElsheikh!)
