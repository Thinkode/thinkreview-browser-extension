# ThinkReview - Version 2.2.6.2 Release Notes

**Release Date:** 22 May 2026

---

## 🎯 What's New

### Clearer version label for this build 🔢

The extension now reports version **2.2.6.2** so what you see in the browser, diagnostics, and release tracking all match this tagged build. Under the hood it is the same revision as **2.2.5**—there are no additional code changes beyond that version line.

### Sign in from the review panel via the ThinkReview portal 🔐

When you need to sign in before running a review, the integrated panel guides you to the ThinkReview portal in a new tab, with a clearer on-page prompt and safer handling from the background script. (Thank you @JElsheikh!)

### Optional announcements in the review panel 📣

Important updates can appear as a small banner at the top of the review panel. You can dismiss a message when you have seen it; your choice is remembered for that announcement so it does not keep coming back. (Thank you @JElsheikh!)

### Agent reviews start sooner and keep working if you step away ⚡

Custom agent reviews start in parallel with the main review flow so agent tabs tend to populate faster, and the panel explains that you may close it while work continues in the cloud. (Thank you @JElsheikh!)

### Clearer summaries in the integrated panel 📋

When the AI writes the main review summary as a numbered list, the panel shows it as an easy-to-scan bullet list so host-site styling is less likely to hide the structure. (Thank you @JElsheikh!)

### Reviews start when you choose (manual-first) 🖱️

Automatic review on page load is turned off in this build so reviews begin when you open ThinkReview or use the trigger on the page, with popup controls hidden to match that behavior for now. (Thank you @JElsheikh!)

### Smoother experience when moving between pull requests 🧭

Switching to another pull request or merge request clears the right parts of the panel state without wiping your context when you navigate away, so returning to ThinkReview on a new change is more predictable. (Thank you @JElsheikh!)

### Consistent “ThinkReview” naming on the page 🏷️

Fallback and in-page labels that still said “AI Review” now read **ThinkReview**, matching the rest of the product. (Thank you @JElsheikh!)
