# ThinkReview - Version 2.2.2 Release Notes

**Release Date:** 24 April 2026

---

## 🎯 What's New

### Clearer published version 🏷️

The extension’s reported version now matches the **2.2.2** release line in the store and in the browser, so support and troubleshooting stay straightforward after the last tagged build.

### Same improvements you get from 2.2.1 📦

This patch follows the **v2.2.1** release with no additional product changes. If you are upgrading from **2.2.0** or earlier, you still get everything from **2.2.1**, including leaner permissions (no in-extension Stripe checkout), smoother **left-side** integrated panel resizing, cleaner **Azure DevOps** review list styling, and more reliable **custom domain** flows in the popup.
