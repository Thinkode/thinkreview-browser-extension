## üìù Getting Started

1. **Install Extension**: Add ThinkReview from Chrome Web Store
2. **Sign In**: Click the extension icon and sign in with Google
3. **Navigate to MR**: Open any GitLab merge request
4. **Start Review**: Click the "AI Review" button to generate code review
5. **Ask Questions**: Use the chat interface to ask specific questions about the code
