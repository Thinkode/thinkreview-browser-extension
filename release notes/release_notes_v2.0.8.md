# ThinkReview - Version 2.0.8 Release Notes

**Release Date:** 02 April 2026

---

## 🎯 What's New

### Manual or Automatic Review Start for All AI Providers ⚙️

You can now choose how reviews start, no matter which AI provider you use.

- **More control:** The "Start review automatically" setting now works for both ThinkReview Cloud and Ollama.
- **Manual mode support:** Prefer to start reviews only when needed? You can now keep auto-start off for any provider.
- **Clearer guidance:** Updated in-app tooltips now explain exactly what each mode does.

---

### Better Large-PR Upgrade Guidance for Free Users 📈

When a pull request is too large for the free plan, ThinkReview now shows a clear upgrade prompt instead of a generic failure experience.

- **Clear limit messaging:** You now see when PR size is above the free-plan threshold.
- **Actionable next step:** The upgrade flow explains how to continue reviewing larger PRs.
- **Smoother experience:** The review panel now recovers cleanly when a later request succeeds.

---

## 🔧 Additional Improvements

- Added a new **Settings** button directly in the integrated review panel header for quicker access to extension settings.
- Refined the bug-report action in the review header to a cleaner icon-based interaction with tooltip support.
- Improved formatting behavior for markdown/code rendering in review output and suggestion blocks.
