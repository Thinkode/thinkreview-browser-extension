# ThinkReview - Version 2.3.1 Release Notes

**Release Date:** 26 June 2026

---

## 🎯 What's New

### Full Context reviews from the extension popup 🔗

A new **Full Context** shortcut in the popup takes you straight to **integrations** on the ThinkReview portal, where you can set up richer reviews that go beyond diff-only analysis. Open it when you want to connect repositories and enable deeper context for your code reviews.

### MCP setup shortcut in the popup 🧩

The popup now includes an **MCP** button that opens **MCP setup** on the ThinkReview portal in one click. Configure Model Context Protocol integrations from the portal without hunting through menus.

### Bring Your Own Key (BYOK) moving to the portal 🔑

OpenRouter setup has been removed from the extension popup for now. A clear notice in the **AI Provider** section explains that **Bring Your Own Key** options will be managed on the [ThinkReview portal](https://portal.thinkreview.dev) soon, so you know where to configure custom API keys going forward.

### Cleaner portal shortcuts in the popup ✨

Popup shortcuts have been refreshed: **Full Context** and **MCP** are easier to find, and older **New** badges on **Agents** and **Usage** have been removed so the menu stays focused on what matters most.
