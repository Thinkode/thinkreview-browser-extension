# ThinkReview - Version 2.0.9 Release Notes

**Release Date:** 04 April 2026

---

## 🎯 What's New

### Smarter Auto-Review Triggering for Large Changes 🧠

ThinkReview now makes a smarter decision before auto-starting a review, especially on larger pull requests.

- **Improved auto-review control:** Auto-triggered reviews can now be skipped for very large diffs.
- **Better credit management:** This helps prevent unnecessary review-credit usage on changes that are better handled manually.
- **More predictable behavior:** Manual reviews still work as expected when you explicitly start them.

---

### New In-Panel Tip to Switch to Manual Review Mode 💡

When you hit usage limits, ThinkReview now shows a helpful tip in the review panel to guide your next step.

- **Actionable guidance:** A new banner explains how manual mode can help you manage daily review credits.
- **One-click settings access:** The banner includes a direct "Go to Settings" action.
- **Smoother handoff:** Settings can now open focused on the relevant auto-start section.

---

## 🔧 Additional Improvements

- Improved popup settings navigation with deep-link scrolling and visual highlight for important sections.
- Refined setup and documentation links in the popup for clearer onboarding instructions.
- Updated some review and popup internals for cleaner handling of limits and settings flow.
