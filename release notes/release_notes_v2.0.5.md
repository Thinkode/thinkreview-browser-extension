# ThinkReview - Version 2.0.5 Release Notes

**Release Date:** 26 March 2026

---

## 🎯 What's New

### Better Azure DevOps URL Handling for Some Accounts 🔎

ThinkReview now identifies Azure DevOps project context more reliably for users on `*.visualstudio.com` accounts.

- **More accurate project detection:** Review context is now resolved correctly for legacy Visual Studio-hosted Azure DevOps URLs.
- **Improved review reliability:** Fewer cases where ThinkReview could not map the current repository/project correctly.
- **Smoother experience across Azure URL formats:** Detection behavior now better matches different Azure DevOps host patterns.

---

## 🔧 Additional Improvements

- Refined internal URL parsing logic for Azure DevOps project extraction.
- Improved handling of edge cases where project names could be inferred incorrectly.
