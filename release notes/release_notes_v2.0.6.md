# ThinkReview - Version 2.0.6 Release Notes

**Release Date:** 26 March 2026

---

## 🎯 What's New

### Code Suggestions Tab Now Shows Suggestion Count 🔢

The Code Suggestions tab now shows how many suggestions were generated for your review.

- **Instant visibility:** You can quickly see the number of code suggestions without opening the tab first.
- **Clearer prioritization:** Counts help you decide when to jump into suggestions during larger reviews.
- **Cleaner tab behavior:** The old "New" badge has been replaced with a more informative count indicator.

---

### Smarter Timing for Feedback Prompt ⭐

The feedback/rating prompt is now shown based on your total completed reviews, not only daily activity.

- **More consistent prompting:** The prompt appears after overall usage milestones instead of requiring many reviews in one day.
- **Better fit for regular users:** Users who review steadily across days are now more likely to see the prompt at the right time.
- **Same cooldown behavior:** "Maybe Later" still respects the existing cooldown before showing again.

---

## 🔧 Additional Improvements

- Improved daily-limit upgrade experience with clearer plan presentation and pricing details.
- Added support for remote-configured upgrade messaging and promotions in the upgrade prompt.
- Polished integrated review panel behavior for upgrade-state scrolling and layout consistency.
