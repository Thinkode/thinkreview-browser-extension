# ThinkReview - Version 2.0.7 Release Notes

**Release Date:** 01 April 2026

---

## 🎯 What's New

### New Agent Review Tabs in the Integrated Panel 🤖

ThinkReview now supports custom AI Agents directly in your review panel, so you can run focused checks alongside the standard review.

- **Dedicated agent tabs:** Each configured agent gets its own tab and output area in the integrated review panel.
- **Clear processing states:** Agent results now show clear loading, still-processing, skipped, and error states so you always know what is happening.
- **Smarter empty state:** If you do not have any agents configured yet, ThinkReview shows a guided "Agents" placeholder with a direct link to set them up.

---

### New "Agents" Shortcut in the Popup ✨

You can now open your agent setup page directly from the extension popup.

- **Faster access:** A new Agents button in the popup takes you straight to the agents portal.
- **Quicker onboarding:** Easier path to create and enable custom reviewers for future code reviews.

---

## 🔧 Additional Improvements

- Improved agent response rendering for cleaner section formatting and better readability.
- Added per-item copy interactions in agent result sections for easier sharing and follow-up.
- Improved safety when rendering skipped-agent reasons by sanitizing content before display.
